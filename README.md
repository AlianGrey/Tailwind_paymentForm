# React + TypeScript + Vite
Simple Payment Form


A simple payment form built with React, styled with Tailwind CSS, and bundled using Vite. The project uses Bun as the development server and package manager for fast and modern workflows.

Features

Responsive and mobile-friendly form

Atomic styling with Tailwind CSS

Easy to extend with new form fields or payment methods

Fast development server with Bun + Vite
git
Technologies Used

React – UI library for building components

Tailwind CSS – atomic CSS framework for styling

Vite – fast bundler and dev server

Bun – lightweight runtime and package manager

Getting Started
1. Clone the repository
git clone https://github.com/username/simple-payment-form.git
cd simple-payment-form

2. Install dependencies
bun install

3. Run the development server
bun dev


Open http://localhost:5173
 in your browser.
